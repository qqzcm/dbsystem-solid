## 运行方式

### 方式一：使用 Java 命令运行
```bash
java -jar app/target/app-1.0.0.jar
```

### 方式二：指定端口运行
```bash
java -jar app/target/app-1.0.0.jar --server.port=8080
```

### 方式三：使用 Maven 运行（开发模式）
```bash
cd app
mvn spring-boot:run
```

## 部署说明

### 1. 系统要求
- Java 11 或更高版本
- 至少 512MB 可用内存（推荐 1GB+）

### 2. 部署步骤
1. 将 `app-1.0.0.jar` 复制到目标服务器
2. 确保服务器已安装 Java 11+
3. 运行命令：
   ```bash
   java -jar app-1.0.0.jar
   ```
4. 访问应用：`http://localhost:8080`

### 3. 后台运行（Linux）
```bash
nohup java -jar app-1.0.0.jar > app.log 2>&1 &
```

### 4. 作为 Windows 服务运行
可以使用 [NSSM](https://nssm.cc/) 将 JAR 文件注册为 Windows 服务。

## 配置文件

应用配置文件位于 JAR 包内的 `application.yml`，如需修改配置，可以：
1. 解压 JAR 文件修改 `application.yml`
2. 或使用外部配置文件：
   ```bash
   java -jar app-1.0.0.jar --spring.config.location=file:./application.yml
   ```

## 注意事项

1. **端口占用**：默认端口 8080，确保端口未被占用
2. **数据文件**：确保 `data/` 目录下的数据文件可访问
3. **WASM 文件**：确保 `js/nkdvWasm/` 目录下的 WASM 文件可访问
4. **内存设置**：如果遇到内存不足，可以增加 JVM 内存：
   ```bash
   java -Xmx2g -jar app/target/app-1.0.0.jar
   ```
